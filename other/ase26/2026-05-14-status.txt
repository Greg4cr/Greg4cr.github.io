How to restore daily updates
----------------------------
Any one of the following will let the next run produce fresh reports:

  (a) Re-enable computer-use (Settings → Desktop app → Computer use)
      so the task can drive the logged-in HotCRP browser session, the
      same way earlier successful runs did.
  (b) Save a HotCRP API token (or a freshly exported PC CSV containing
      name, email, tags, and review completion columns) into the
      project folder so the task can read it locally.
  (c) Manually export the PC CSV from HotCRP and drop it in
      /Users/ggay/Claude/Projects/ase-26-monitoring/ before the next
      scheduled run.

Then re-run the daily-progress-update scheduled task.
